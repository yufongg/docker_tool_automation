# Prerequisites:
1. Add user to docker group              
2. Change file permission to make it executable

# Tools used:
1. Dockle
2. Trivy
3. Snyk monitor

# What it does:
It automatically installs the 3 tools and run scan on local or pull image from dockerhub
Also, it saves output to a json file.
